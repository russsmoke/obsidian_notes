Эта формула решает проблему: если мы просто сложим мощности множеств, то элементы, принадлежавшие одновременно двум множествам, будут учтены дважды (или более раз). Формула позволяет компенсировать эти повторные подсчеты.

## Размещения, перестановки, сочетания без повторений
### Размещение
Размещением из n элементов по m элементов ($0<m<=n$) называется любое упорядоченное подмножество n данного множества, состоящее из m элементов.
Это комбинации, состоящие из m элементов, которые отличаются друг от друга составом элементов, либо порядком их расположения.

![[Pasted image 20260301150120.png]]


> [!NOTE] Замечание
> Комбинаторика классифицирует любой набор элементов по двум признакам:
> - Состав элементов (какие именно предметы взяли)
> - Порядок элементов (стоит ли этот предмет первым, вторым и т.д.)

Для размещения важно и то, и другое.

> [!NOTE] Как запомнить определение размещения
> Когда формируется компания из k человек и их сажают на k стульев, то:
> - Если поменять двух людей местами (порядок), это будет другая рассадка
> - Если заменить одного человека на другого (состав), это будет другая рассадка

### Перестановки
Перестановками из n элементов называется размещение из n элементов по n.
Это комбинации, состоящие из n элементов и отличающиеся друг от друга только порядком следования элементов. (состав всегда одинаковый)

![[Pasted image 20260301151203.png]]

### Сочетание
Сочетанием из n элементов по m ($0<m<=n$) элементов называется любое подмножеств, которое содержит m элементов данного множества.
Это комбинации из m элементов, составленных из данных n элементов, которые отличаются друг от друга хотя бы одним элементом. Сочетания отличаются только составом элементов, порядок следования элементов не учитывается.
