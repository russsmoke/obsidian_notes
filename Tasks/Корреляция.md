**Статистическая мера**, которая показывает, насколько сильно связаны между собой ***две*** переменные.
Помогает понять, как изменение одного показателя влияет на другой.

## Виды корреляции:
- По направлению: Положительная (две переменные взаимно увеличиваются), отрицательная (одна переменная увеличивается, другая уменьшается), нулевая (не демонстрируют связи).
- По форме зависимости: линейная (аппроксимируется прямой линией), нелинейная (связь описывается кривой)
- По количеству исследуемых переменных: парная (две переменные), множественная (три и более), частная (две переменные в вакууме)

## Методы измерения
Основной инструмент измерения - ***коэффициент корреляции.*** Принимает значения от -1 до +1.

## Разновидности
Универсального коэффициента корреляции не существует, выбор зависит от природы переменных - **количественных, ранговых или категориальных**.
1) ***Коэффициент Пирсона (r)*** - для **количественных** данных с нормальным распределением. Измеряет силу и направление линейной связи между двумя переменными.

2) ***Коэффициент Спирмена ($\rho$)*** - для **ранговых** данных, где значения можно упорядочить, но интервалы между неравномерны. Устойчив к выбросам, если распределение данных не является нормальным. 
3) ***Коэффициент Кендалла ($T$)*** - для **ранговых или порядковых** данных, учитывает количество согласованных и несогласованных пар наблюдений. Метод более консервативен, чем метод Спирмена. Для небольших выборок.
4) ***Фи-корреляция ($\phi$)*** - для  **бинарных** переменных (да/нет, 0/1).
