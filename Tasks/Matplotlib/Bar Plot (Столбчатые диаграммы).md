# Description
Визуализация данных в виде столбцов, где каждый столбец представляет собой определенную группу или категорию, а длина или высота столбца соответствует числовому значению или количеству.

Две оси: ось категорий и ось значений

Столбчатые диаграммы используются для сравнения значений данных в разных категориях или группах. Позволяют легко определить, какие категории имеют более высокие или более низкие значения. Помогают визуализировать разброс или концентрацию данных внутри каждой группы. Ранжирование позволяет выделить наиболее или наименее значимые категории.

`plt.bar(x, height, width=0.8, bottom=None, *, align='center', data=None, **kwargs)`
- x: координаты Х столбцов
- height: высоты столбцов
- width: ширина столбцов
- bottom: координаты нижней стороны столбцов
- align: выравнивание столбцов по координатам х. "center" - центрирование по х, "edge" - выравнивание левого края по х. Чтобы выровнять по правому краю, указывается отрицательная ширина и align="edge"

## Горизонтальные столбчатые диаграммы
Применяется функция `plt.barh(y, width, height=0.8, left=None, *, align="center", data=None, **kwargs)`
- y: координаты Y столбцов
- width: ширина столбцов
- height: высоты столбцов
- left: координаты левой стороны столбцов
- align: выравнивание по координатам х. "center" - центрирование по y, "edge" - выравнивание верхнего края по Y. Чтобы выровнять столбцы по верху, указывается отрицательная высота и align="edge"
